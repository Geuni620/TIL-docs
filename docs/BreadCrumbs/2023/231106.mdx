### PWA

- PWA 예약알림 기능을 추가해줬다.
  - 크롬(Mac) / 사파리(Iphone) 두 기기를 이용해서 임시 페이지에 PWA 알림을 구독해줬다
  - 구독 후, node-schedule를 이용해서 일정시간을 지정해주고, 알림이 가는지를 테스트했다.
  - 크롬에선 잘 적용되는데 반해, 사파리에선 에러가 났는데, 이유를 확인해보니, DB에 구독정보를 저장할 때 endpoint가 db type에 의해 짤린 것이었다.
  - 그래서 db.text 타입으로 변경해준 뒤, 확인해보니 알림이 잘 뜬다. 🙌

### Toggle Swiper

- 기획서를 받고, 구조를 설계했는데, 추후 개발을 진행하다가 기획이 변경됐다.
  - 기존에 route로 구분하던걸 Swiper로 변경하게 됐다.
  - 즉 공개와 비공개를 클릭했을 때 공개된 기록으로 이동 / 비공개 된 기록으로 이동이었는데, 이걸 Swiper로 변경했다.
  - 구조를 다시 바꾸는 작업을 하는데, 여간 까다롭다.
  - 즉, 구조화를 똑바로하지 못했다는 반증인 것 같다.
