### Typescript

```TSX
export const WORKLIFE = [
  {
    id: 1,
    image: '/image/worklife/ingjae.svg',
    title: '나만의 제조도 캐릭터',
    description: '제조도 잉재라이브러리',
    // link: null,
  },
  {
    id: 2,
    image: '/image/worklife/nanjang.png',
    title: '자유로운 난장토론',
    description: '제조도 난장독대',
  },
];
```

- 이 경우엔 링크가 들어가는 경우도 있고 들어가지 않는 경우도 있다.
- 링크가 들어가야하는 경우엔 링크를 기입하고, 아닐 경우엔 null처리했다

- 기존에는 link가 없을 경우 null 값으로 처리했다

<br />

- 하지만 이번엔 link가 들어가는 경우만 기입하고, 나머지는 제거한 후 type을

```TSX
interface Item {
  id: number;
  title: string;
  description: string;
  image: string;
  link?: string;
}
```

- 옵셔널 체이닝을 통해서 기입해주었다.

<br />

- 옵셔널 처리하고 나니, 이건 서버에서 예측하지 못하는게 아니라 예측가능한 null이라는 생각이 들어 다시 변경해주었다.

```TSX
interface Item {
  id: number;
  title: string;
  description: string;
  image: string;
  link: string | null;
}
```

```TSX
export const WORKLIFE = [

    id: 1,

    title: '-',
    description: '-',
    link: 'https://',
  },
];
```
